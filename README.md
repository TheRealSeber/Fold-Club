# Fold Club

Source code for an e-commerce store selling (also) viral low-poly papercraft masks.

**First rule of Fold Club: You stick it together.**

## About

Fold Club is a full-stack e-commerce application built with SvelteKit and Bun, offering a unique shopping experience for low-poly papercraft masks and related products.

## Tech Stack

- **Frontend**: SvelteKit
- **Runtime**: Bun
- **Full Stack**: SvelteKit with Bun

## Getting Started

```bash
# Install dependencies
bun install

# Run development server
bun run dev

# Build for production
bun run build

# Preview production build
bun run preview
```
